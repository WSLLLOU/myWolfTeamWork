# 背景介绍
场地的四个顶点 `D1-D4` 为哨岗区. 红蓝双方各拥有对角线的两个哨岗区. 哨岗区提供了哨岗感知系统的安装接口, 参赛队伍可安装哨岗感知系统.   
`D2、D4` 为 `红方` 哨岗区域, `D1、D3` 为 `蓝方` 哨岗区域, 哨岗固定哨岗感知系统的安装高度为距离地面 `1768 mm` .
  
<br>
